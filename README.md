# easy-report

## 设计理念

提高工作效率，在打工人的世界里移除日报和周报

## 功能概述

1. 任务列表
    * 可根据标签调整分组
    * 可调整优先级
2. 生成日报/周报
    * 根据生成配置将本日/本周内容汇总，生成日报/周报
    * 自动关联 git
3. 生成甘特图
4. Copy/发送邮箱
5. GPT4.0 润色、总结
6. 工作台/看板
7. 插件扩展

## 技术架构

MAUI Blazor跨端应用，支持在线和离线模式，离线时本地采用Sqlite缓存，在线时双向同步

暂时以 `Windows-Local` 为准

## 概要设计

此设计大部分为 `v1.0` 要包含的大概功能，标记为 `v2.x` 则为设计中的功能.

> `v1.0` 版本将会主打 `windows` 下的离线版本

### 模块

#### 工作台/看板

1. 甘特图面板
2. 任务看板
3. 快速入口
4. 延时任务追踪

#### 任务列表

1. 增删改查任务(可设置任务预安排日期)
2. 调整任务优先级
3. 调整任务[分类/标签]
    * 添加[分类/标签]快速入口
4. 可视化动态甘特图
    * 可交互甘特图
        * 可以根据[分类/标签]筛选
        * 当前日期范围展示
    * 选取可看具体描述
5. 任务可链接项目 `v2.x`
6. 任务可链接外部链接

#### 日报/周报

1. 日报/周报生成(Markdown/Text/Html)
2. GPT4.0 润色 `v2.x`
3. 提交方式(Copy/发送邮件)
4. 关联 `git` `v2.x`
5. 模板配置 `v1.x`

#### 可变更布局

1. 看板可作为全局布局
2. 其它模块可以拆分/独立/Tab显示

#### 扩展商店 `v3.x`

主要为一些扩展，例如实现了钉钉的发送周报，就可以集成进来. 包括发送到邮箱以及周报美化等等等，还待具体设计.

#### 设置

1. 离线/在线 `v2.x`
2. 同步设置 `v2.x`
3. 配置 `git` 仓库列表 `v2.x`
4. 主题配置(内置明暗两种)
5. 导出/备份
6. 多语言 `v2.x`
7. 优先级配置
8. 分组配置

#### 其它

1. 自然周计算器
